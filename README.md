# Jacky's Cloud Blog

This repository contains the source for [Jacky's Cloud Blog](https://jackychiou.github.io), built with Jekyll and the just-the-docs theme.

## Local Development

Install dependencies:

```powershell
bundle install
```

Run the site locally:

```powershell
bundle exec jekyll serve
```

Build the site:

```powershell
bundle exec jekyll build
```

## Content Structure

- `index.md`: home page
- `docs/`: Azure knowledge base and articles
- `workshop/`: workshop materials
