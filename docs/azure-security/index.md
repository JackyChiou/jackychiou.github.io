# Azure 資安合規

根據 **CIS Microsoft Azure Foundations Benchmark** 整理的設定指南，協助企業達成 Azure 資安基準合規要求。

{: .important }
CIS Benchmark 分為 Level 1（基本）與 Level 2（進階）兩個等級。Level 2 項目通常需要額外授權，但能大幅提升整體安全態勢。
